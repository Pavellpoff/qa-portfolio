# Mobile Testing - Voyager for Lemmy

Проект по тестированию Android-приложения **Voyager for Lemmy** с использованием Android Studio Emulator и перехвата HTTPS-трафика через Proxyman.

Тестируемое приложение:  
https://github.com/aeharding/voyager

Платформа распространения:  
https://f-droid.org/packages/app.vger.voyager/

---

# Описание проекта

В рамках проекта проведено исследовательское и негативное тестирование мобильного приложения Voyager for Lemmy.

Основной фокус:
- анализ API и GUI взаимодействия;
- проверка устойчивости интерфейса;
- тестирование обработки нестандартных данных;
- анализ поведения приложения при ошибках сервера;
- проверка обработки HTML/script-инъекций;
- исследование кэширования и отображения данных.

Для тестирования использовалась подмена и модификация API responses через Proxyman.

---

# Цель проекта

- Проверить устойчивость Android-приложения к некорректным данным

# Основные наблюдения

В ходе тестирования выявлено:

- приложение устойчиво к некорректным API responses;
- интерфейс корректно обрабатывает отсутствующие поля;
- приложение не падает при HTTP 500;
- предположительно используется локальное кэширование данных;
- HTML/script-код не выполняется и отображается как текст;
- UI стабильно работает при длинном контенте и нестандартных данных.
