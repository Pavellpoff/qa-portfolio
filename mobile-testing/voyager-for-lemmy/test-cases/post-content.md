# Обработка изображений поста

## TC-01 — Обработка отсутствующего или невалидного изображения

### Окружение
- Android Studio Emulator
- Android 13 (API 33 “Tiramisu”)
- Resolution: 2400×1800
- 480 dpi
- Proxyman

---

### Проверяемые сценарии
1. Поле `url` содержит невалидное значение
2. Поле `url` содержит пустую строку
3. Поле `url` отсутствует в response

---

### Шаги
1. Открыть пост в приложении
2. Перехватить response:
   ```http
   GET /api/v3/post?id=...
3. Изменить значение поля url согласно сценарию
4. Перезагрузить экран поста

---

### Ожидаемый результат
- приложение не падает
- интерфейс отображается корректно
- структура экрана не нарушается
- пост корректно отображается без изображения или с placeholder

---

### Фактический результат

Во всех сценариях:
- приложение работает стабильно
- структура интерфейса не нарушена
- пост отображается корректно

---

### Наблюдения

При невалидном URL отображается текст: failed to load media 🙁

При пустом или отсутствующем поле url пост отображается без изображения
