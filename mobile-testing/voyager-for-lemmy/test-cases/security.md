- Android Studio  
- API 33 “Tiramisu”, Android 13  
- Resolution: 2400×1800  
- 480 dpi  
- 6,4”  
- Proxyman  

**Шаги:**  
1. Очистить кэш приложения  
2. Перезагрузить пост  
3. Перехватить response:  
   `GET /api/v3/post?id=...`  
4. Добавить в поле `body`:  
   ```html
   <script>alert(1)</script>


**Ожидаемый результат:**

- приложение не падает
- HTML/script-код не выполняется
- интерфейс отображается корректно

**Фактический результат:**

- приложение не падает
- HTML/script-тег отображается как обычный текст
- выполнение script-кода не обнаружено
- интерфейс отображается корректно

Приложение корректно обрабатывает HTML/script-теги и не выполняет встроенный script-код.
