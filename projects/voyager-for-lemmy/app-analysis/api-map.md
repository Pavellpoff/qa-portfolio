# Создание карты приложения

## Endpoint

http
GET https://lemmy.zip/api/v3/post?id=  (напр: 64319097) 


```
| Поле API | Отображение в GUI | Что тестировать                                |
| -------- | ----------------- | ---------------------------------------------- |
| `url`    | изображение поста | битая ссылка, пустое значение, отсутствие поля |
| `name`   | заголовок поста   | длинный текст, отсутствие заголовка            |
| `body`   | текст поста       | длинный текст, эмодзи, спецсимволы             |
```

## Анализ комментариев

GET https://lemmy.zip/api/v3/comment/list?post_id=64319097&sort=Hot&type_=All&limit=60&max_depth=6

```
| Поле API         | Отображение в GUI                         | Что тестировать                                                                |
| ---------------- | ----------------------------------------- | ------------------------------------------------------------------------------ |
| `creator.name`   | имя комментатора                          | длинное имя                                                                    |
| `content`        | текст комментария                         | длинный комментарий, пустой комментарий                                        |
| `comments[]`     | список комментариев                       | пустой массив комментариев                                                     |
| comments loading | отображение комментариев на втором экране | когда появляется запрос: сразу или при скролле; поведение при потере интернета |
```
