# Банковские счета (UI)

## Описание

Данный набор автотестов проверяет функциональность управления банковскими счетами в приложении Cypress Real World App.

## Проверки

### 1. Создание банковского счета

Проверяется:
- успешное создание нового банковского счета;
- отображение созданного счета в списке;
- успешное удаление банковского счета;
- отображение статуса **Deleted** для удаленного счета.

---

### 2. Проверка минимальной длины полей

Проверяется:
- ввод значений меньше минимально допустимой длины для полей **Bank Name** и **Account Number**;
- отображение сообщений валидации;
- блокировка кнопки **Save**;
- невозможность отправить форму.

---

### 3. Проверка максимальной длины полей

Проверяется:
- ввод значения, превышающего максимально допустимую длину поля **Account Number**;
- отображение сообщения валидации;
- блокировка кнопки **Save**;
- невозможность отправить форму.

---
### 4. Проверка минимально допустимых значений

Проверяется:
- ввод минимально допустимого значения в поле **Bank Name**;
- ввод минимально допустимого значения в поле **Account Number**;
- успешная отправка формы;
- создание банковского счета;
- отображение созданного счета в списке.


---
### 5. Проверка валидации поля Routing Number

Проверяется:
- ввод некорректного значения в поле **Routing Number**;
- отображение сообщения валидации;
- блокировка кнопки **Save**;
- невозможность отправить форму.

---
### 6. Проверка длинного значения поля Bank Name

Проверяется:
- ввод длинного значения в поле **Bank Name**;
- успешная отправка формы;
- создание банковского счета;
- отображение созданного счета в списке.

---
### 7. Проверка максимально допустимого значения Account Number

Проверяется:
- ввод максимально допустимого значения в поле **Account Number**;
- успешная отправка формы;
- создание банковского счета;
- отображение созданного счета в списке.


---
### 8. Проверка полей Routing Number и Account Number, состоящих только из пробелов

Проверяется:
- ввод значений, состоящих только из пробелов, в поля **Routing Number** и **Account Number**;
- успешная отправка формы;
- создание банковского счета;
- отображение созданного счета в списке.
