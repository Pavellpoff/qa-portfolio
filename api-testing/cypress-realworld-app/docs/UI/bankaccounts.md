# Банковские счета (UI)

## Описание

Данный набор автотестов проверяет функциональность управления банковскими счетами в приложении Cypress Real World App.

---

## Покрываемые сценарии

| № | Сценарий | Статус |
|---|----------|:------:|
| 1 | Создание банковского счёта | ✅ |
| 2 | Удаление банковского счёта | ✅ |
| 3 | Валидация минимальной длины полей | ✅ |
| 4 | Валидация максимальной длины полей | ✅ |
| 5 | Валидация обязательных полей | ✅ |

---

## Используемые возможности Cypress

- `cy.loginpavel()`
- `cy.get()`
- `cy.click()`
- `cy.type()`
- `cy.contains()`
- `should()`
- `cy.task()`

---

## Проверки

### Создание банковского счёта

Проверяется:

- успешное создание нового банковского счёта;
- отображение созданного счёта в списке;
- удаление банковского счёта;
- изменение статуса удалённого счёта.

---

### Проверка минимальной длины полей

Проверяется:

- невозможность отправить форму;
- отображение сообщений валидации;
- блокировка кнопки **Save**.

---

### Проверка максимальной длины полей

Проверяется:

- невозможность отправить форму;
- отображение сообщений валидации;
- проверка верхних границ длины полей.

---

### Проверка обязательных полей

Проверяется:

- отображение сообщений об обязательных полях;
- невозможность отправить пустую форму.
