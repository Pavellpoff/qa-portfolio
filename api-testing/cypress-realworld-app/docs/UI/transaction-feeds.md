# Лента транзакций (UI)

## Описание

Набор автотестов проверяет работу ленты транзакций, навигации между разделами и фильтрации данных.

---

## Функциональные сценарии

| № | Сценарий | Статус |
|---|----------|:------:|
| 1 | Навигация между разделами транзакций | ✅ |
| 2 | Фильтрация по диапазону дат | ✅ |
| 3 | Сброс фильтра по дате | ✅ |
| 4 | Фильтрация по диапазону суммы | ✅ |
| 5 | Сброс фильтра по сумме | ✅ |
| 6 | Пустой результат фильтра (выбрать диапазон дат, в котором нет ни одной транзакции) | ✅ |
| 7 | Граничные значения суммы (1000–1000) | ❌ |
| 8 | Комбинированная фильтрация | ❌ |
| 9 | Сохранение фильтра после обновления страницы | ❌ |

---

## Проверки

### 1. Проверка навигации по транзакциям и фильтрации по диапазону дат

**Проверяется:**
- создание тестовой транзакции;
- переход между вкладками **Public**, **Contacts** и **Personal**;
- фильтрация личных транзакций по диапазону дат;
- отображение только подходящих транзакций;
- сброс фильтра;
- восстановление полной ленты транзакций.

---
### 2. Проверка фильтрации транзакций по диапазону суммы

**Проверяется:**
- применение фильтра по диапазону суммы;
- отображение транзакций, соответствующих выбранному диапазону;
- сброс фильтра;
- восстановление диапазона суммы по умолчанию.

---
### 3. Проверка отображения пустого результата при отсутствии транзакций

**Проверяется:**
- применение фильтра по диапазону дат;
- отсутствие транзакций за выбранный период;
- отображение сообщения **No Transactions**.
