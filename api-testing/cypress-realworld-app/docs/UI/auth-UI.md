# Авторизация (UI)

## Описание

Данный набор автотестов проверяет работу авторизации пользователя в приложении Cypress Real World App.

---

## Покрываемые сценарии

| № | Сценарий | Статус |
|---|----------|:------:|
| 1 | Успешная авторизация | ✅ |
| 2 | Выход из системы | ✅ |
| 3 | Пустые поля | ✅ |
| 4 | Неверный пароль | ✅ |
| 5 | Неверный логин | ✅ |
| 6 | Неверный логин + пароль | ✅ |
| 7 | Логин с пробелами | ✅ |
| 8 | Пароль с пробелами | ✅ |
| 9 | SQL Injection | ✅ |
| 10 | Enter вместо кнопки | ✅ |

## Проверки

### 1. Positive Auth

**Проверяется:**

- открытие страницы авторизации;
- ввод корректных учетных данных;
- успешная отправка запроса;
- код ответа 200;
- успешный вход пользователя;
- отображение имени пользователя;
- успешный выход из системы.

---

### 2. Пустые поля

**Проверяется:**

- обязательность заполнения логина;
- обязательность заполнения пароля;
- отображение сообщений валидации.

---

### 3. Неверный пароль

**Проверяется:**

- отправка запроса авторизации;
- получение ответа 401;
- отображение сообщения об ошибке.

---
### 4. Неверный логин


**Проверяется:**

- ввод несуществующего имени пользователя;
- отправка запроса авторизации;
- получение HTTP 401;
- отображение сообщения Username or password is invalid.


---
### 5. Неверный логин и пароль

**Проверяется:**

- ввод несуществующего Username;
- ввод неверного Password;
- отправка запроса авторизации;
- получение HTTP 401;
- отображение сообщения Username or password is invalid.


---
### 6. Логин с пробелами

**Проверяется:**

- ввод Username с пробелами в конце;
- ввод корректного Password;
- отправка запроса авторизации;
- получение HTTP 401;
- отображение сообщения Username or password is invalid.


---
### 7. Пароль с пробелами

Проверяется:
- ввод корректного Username;
- ввод Password с пробелами в конце;
- отправка запроса авторизации;
- получение HTTP 401;
- отображение сообщения `Username or password is invalid`.


---
### 8. SQL Injection

Проверяется:
- ввод SQL-инъекции в поле Username;
- ввод произвольного Password;
- отправка запроса авторизации;
- получение HTTP 401;
- отображение сообщения `Username or password is invalid`.

---
### 9. Авторизация по нажатию клавиши Enter

Проверяется:
- ввод корректного Username;
- ввод корректного Password;
- отправка формы нажатием клавиши **Enter**;
- получение HTTP 200;
- успешная авторизация;
- отображение имени пользователя.
