# Создание запроса на перевод денежных средств (UI)

## Описание

Набор автотестов проверяет создание нового запроса на перевод денежных средств между пользователями.

---

|---|----------|:------:|
| 1 | Создание нового запроса на перевод денежных средств | ✅ |
| 2 | Запрос на перевод нулевой суммы | ✅ |
| 3 | Запрос на перевод отрицательной суммы | ✅ |
| 4 | Запрос на перевод дробной суммы | ✅ |

| 7 | Запрос на перевод суммы с пробелом в описании | ✅ |
| 8 | Отмена создания запроса на перевод | ✅ |

## Проверки

### 1. Создание нового запроса на перевод

Проверяется:

- успешная авторизация;
- переход к созданию перевода;
- выбор получателя;
- ввод суммы;
- ввод описания;
- успешная отправка формы;
- отображение подтверждения создания запроса.



### 2. Создание запроса на перевод с нулевой суммой

**Проверяется:**
- возможность создания запроса с суммой `0`;
- успешная отправка формы;
- отображение сообщения **Transaction Submitted!**;
- корректное отображение созданного запроса с суммой `$0.00`.


### 3. Создание запроса на перевод с отрицательной суммой

**Проверяется:**
- возможность создания запроса с отрицательной суммой;
- успешная отправка формы;
- отображение сообщения **Transaction Submitted!**;
- корректное отображение созданного запроса с отрицательной суммой.


### 4. Создание запроса на перевод с дробной суммой

**Проверяется:**
- возможность создания запроса с дробной суммой;
- успешная отправка формы;
- отображение сообщения **Transaction Submitted!**;
- отображение суммы после создания запроса.



### 5. Создание запроса на перевод на сумму, превышающую доступный баланс

**Проверяется:**
- возможность создания запроса на сумму, превышающую доступный баланс пользователя;
- успешная отправка формы;
- отображение сообщения **Transaction Submitted!**;
- отображение созданного запроса.



### 6. Проверка обязательности поля Description

**Проверяется:**
- невозможность создания запроса с пустым полем **Description**;
- блокировка кнопки **Request**;
- отображение сообщения **Please enter a note**.



### 7. Создание запроса на перевод с описанием, состоящим только из пробелов

**Проверяется:**
- возможность создания запроса с описанием, содержащим только пробельные символы;
- успешная отправка формы;
- отображение сообщения **Transaction Submitted!**;
- отображение созданного запроса.



### 8. Проверка отмены создания запроса на перевод

**Проверяется:**
- ввод данных без отправки формы;
- сброс состояния формы после обновления страницы;
- возврат к первому шагу создания запроса;
- отсутствие незавершенного запроса в истории транзакций.
