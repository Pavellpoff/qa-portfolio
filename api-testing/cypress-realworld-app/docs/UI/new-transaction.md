# Создание денежных переводов (UI)

## Описание

Набор автотестов проверяет создание нового денежного перевода между пользователями.

---

## Покрываемые сценарии

| № | Сценарий | Статус |
|---|----------|:------:|
| 1 | Создание нового запроса на перевод денежных средств | ✅ |
| 2 | Перевод нулевой суммы | ❌ |
| 3 | Перевод отрицательной суммы | ❌ |
| 4 | Перевод дробной суммы | ❌ |
| 5 | Перевод суммы, которой у пользователя нет на счете | ❌ |
| 6 | Перевод суммы с пустым описанием | ❌ |
| 7 | Перевод суммы с максимальной длиной описания | ❌ |
| 8 | Перевод суммы с только пробелами в описании | ❌ |
| 9 | Отмена создания перевода | ❌ |


---

## Проверки

### Создание нового запроса на перевод

Проверяется:

- успешная авторизация;
- переход к созданию перевода;
- выбор получателя;
- ввод суммы;
- ввод описания;
- успешная отправка формы;
- отображение подтверждения создания запроса.
