# Личный кабинет (UI)

## Описание

Данный набор автотестов проверяет работу формы редактирования профиля пользователя.

---

## Покрываемые сценарии

| № | Сценарий | Статус |
|---|----------|:------:|
| 1 | Позитивный сценарий изменения профиля | ✅ |
| 2 | Проверка обязательных полей | ✅ |
| 3 | Проверка валидации Email | ✅ |
| 4 | Проверка валидации номера телефона | ✅ |
| 5 | Проверка граничных случаев ввода | ✅ |
| 6 | Проверка отмены изменений (изменить = не сохранять = обновить страницу) | ❌ |




---

## Проверки

### 1. Позитивный сценарий изменения профиля

**Проверяется:**
- переход в раздел **User Settings**;
- изменение данных профиля;
- успешное сохранение изменений;
- отображение обновленного имени пользователя;
- сохранение изменений после обновления страницы.


### 2. Проверка обязательных полей

Проверяется:

- очистка обязательных полей;
- отображение сообщений валидации;
- блокировка кнопки сохранения.

---

### 3. Проверка Email и Phone Number

Проверяется:

- валидация некорректного Email;
- валидация номера телефона;
- невозможность сохранить форму.

---

### 4. Проверка граничных случаев

Проверяется:

- использование чисел вместо имени;
- ввод длинной строки в поле фамилии;
- ввод длинного номера телефона;
- сохранение введённых данных.
