# 📌 Информация о проекте Cypress Real World App

## Цель проекта:

Данный проект создан для демонстрации моих практических навыков в области QA Automation на примере приложения Cypress Real World App.

В рамках проекта разрабатываются UI- и API-автотесты, оформляются тест-кейсы, баг-репорты, результаты исследовательского тестирования и анализ найденных дефектов.

Основная цель проекта - показать не только умение писать автотесты, но и полный процесс тестирования: от проектирования тестов до анализа обнаруженных проблем.


## Статус проекта

На текущий момент автоматизирована часть функциональности приложения. Для реализованных модулей подготовлены:

- UI- и API-автотесты;
- тест-кейсы;
- наблюдения (Findings);
- баг-репорты;
- документация по каждому модулю.

Часть сценариев пока остается в статусе запланировано. Они сознательно сохранены в проекте, чтобы показать текущее состояние покрытия и дальнейший план развития, аналогично как это происходит в реальных коммерческих проектах.

## Подход к проекту

Проект не стремится показать "идеальное" покрытие всей функциональности приложения.

Задача - продемонстрировать реальный процесс работы QA Engineer.

Поэтому репозиторий включает не только завершенные автотесты, но и сценарии, которые уже проанализированы и запланированы к автоматизации.

# 📝 Документация по проекту

В рамках проекта подготовлена тестовая документация для реализованных модулей:

-  Test Cases;
- Test Findings;
- Bug Reports;
- Root Cause Analysis.

| Документ            | Назначение                                                             |
| ------------------- | ---------------------------------------------------------------------- |
| Test Cases          | Описание тестовых сценариев UI и API                                   |
| Test Findings       | Наблюдения, требующие уточнения требований или дополнительного анализа |
| Bug Reports         | Подтвержденные дефекты, обнаруженные в ходе тестирования               |
| Root Cause Analysis | Анализ предполагаемой причины одного из критических дефектов           |
