# UI тест-кейсы

В документе представлены тест-кейсы для UI-тестирования приложения Cypress Real World App.

---

## Содержание
 - TC модуля Авторизация (AUTH) = 9 шт.
 - ТС модуля Банковские счета (BANK) = 15 шт.
 - ТС модуля Личный кабинет = 5 шт.
 - ТС модуля Создание запросов на денежный перевод (REQUEST) = 8 шт.
 - ТС Полные сценарии по созданию и подтверждению запросов на денежный перевод (REQUEST. E2E) = 1 шт.
 - ТС модуля Лента транзакций (FEED) = 7 шт.

**Модуль:** Авторизация

**Предусловия**

- Страница `/signin` открыта.

**Шаги**

3. Нажать кнопку **Sign In**.
